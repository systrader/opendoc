# 골든돔 (GoldenDom) 매매 활용 가이드

---

## 1. 신호 개념

골든돔은 **선형회귀선(Linear Regression)**을 기반으로 시장의 국면을 6단계로 분류하고, 다양한 기술적 요인을 점수화하여 매매 신호를 생성하는 시스템입니다.

### 선형회귀선이란?
과거 N개 봉의 가격 데이터에 가장 잘 맞는 직선을 통계적으로 계산한 것입니다. 이동평균선보다 **추세의 방향과 강도**를 더 명확하게 보여줍니다.

### 핵심 특징
- **6단계 국면 분류**: 시장 상태를 6가지로 구분
- **점수 시스템**: -10점~+10점 범위의 정량적 평가
- **3가지 모드**: 종목 유형에 맞는 최적화된 파라미터

---

## 2. 6단계 국면 시스템

차트 왼쪽 상단에 현재 국면이 표시됩니다.

| 국면 | 이름 | 색상 | 조건 | 매매 전략 |
|------|------|------|------|----------|
| **1** | 급등세 | 🔴 빨강 | 상승추세 + 강한기울기 + 고점수 | **핵심 매수 구간** |
| **2** | 상승세 | 🟠 주황 | 상승추세 + 약한기울기 | 보유 유지 |
| **3** | 급락세 | 🔵 파랑 | 하락추세 + 강한기울기 + 저점수 | **즉시 청산** |
| **4** | 하락세 | ⚫ 회색 | 하락추세 + 약한기울기 | 관망, 매수 금지 |
| **5** | 박스상단 | 🟢 초록 | 횡보 + 회귀선 위 | 익절 고려 |
| **6** | 박스하단 | 🟡 노랑 | 횡보 + 회귀선 아래 | 매수 대기 |

### 국면 전환 시나리오

```
이상적인 매수 사이클:
[6 박스하단] → [2 상승세] → [1 급등세] → 매수 진입!

이상적인 청산 사이클:
[1 급등세] → [5 박스상단] → [2 상승세] → [4 하락세] → 청산!

위험 신호:
[1 급등세] → [3 급락세] → 즉시 손절!
```

## 3. 점수 시스템

### 3.1 점수 구조

총점은 **상승점수 - 하락점수**로 계산됩니다.

| 구분 | 범위 | 의미 |
|------|------|------|
| 상승점수 | 0~10점 | 상승 요인의 합 |
| 하락점수 | 0~10점 | 하락 요인의 합 |
| **총점** | -10~+10점 | 최종 판단 기준 |

### 3.2 상승 점수 요인 (최대 10점)

| 요인 | 조건 | 점수 |
|------|------|------|
| 고점 상승 | 최근 고점 > 이전 고점 | +2점 |
| 저점 상승 | 최근 저점 > 이전 저점 | +2점 |
| 강한 양의 기울기 | 기울기 > 0.15% | +3점 |
| 약한 양의 기울기 | 0 < 기울기 ≤ 0.15% | +1점 |
| 강한 상승추세 | 5봉 이상 연속 상승 | +3점 |
| 보통 상승추세 | 3~4봉 연속 상승 | +2점 |
| 약한 상승추세 | 1~2봉 상승 | +1점 |

### 3.3 하락 점수 요인 (최대 10점)

| 요인 | 조건 | 점수 |
|------|------|------|
| 고점 하락 | 최근 고점 < 이전 고점 | +2점 |
| 저점 하락 | 최근 저점 < 이전 저점 | +2점 |
| 강한 음의 기울기 | 기울기 < -0.15% | +3점 |
| 약한 음의 기울기 | -0.15% ≤ 기울기 < 0 | +1점 |
| 강한 하락추세 | 5봉 이상 연속 하락 | +3점 |
| 보통 하락추세 | 3~4봉 연속 하락 | +2점 |
| 약한 하락추세 | 1~2봉 하락 | +1점 |

### 3.4 점수 해석 가이드

| 총점 범위 | 해석 | 행동 |
|----------|------|------|
| +7 ~ +10 | 매우 강한 상승 | 적극 매수, 홀딩 |
| +4 ~ +6 | 강한 상승 | 매수 진입 가능 |
| +1 ~ +3 | 약한 상승 | 매수 대기, 관망 |
| -2 ~ 0 | 중립/약세 | 관망, 신규 매수 금지 |
| -3 ~ -5 | 약한 하락 | 보유 시 청산 준비 |
| -6 ~ -10 | 강한 하락 | 즉시 청산 |

## 4. 3가지 모드 상세

차트 범례 드롭다운에서 모드를 선택할 수 있습니다.

### 4.1 대형주 모드 (MODE 0)

**적용 대상**
- 삼성전자, SK하이닉스 등 시총 10조+ 초대형주
- KOSPI, KOSDAQ 지수
- ETF

**파라미터 설정**

| 항목 | 값 | 설명 |
|------|-----|------|
| 최소상승일 | 2봉 | 짧은 기간도 허용 |
| 변동성 상한 | 22% | 넓은 변동성 허용 |
| 손절률 | -4.0% | 넉넉한 손절폭 |
| 익절률 | +6.5% | 큰 수익 추구 |
| 점수 기준 | 6점 | 낮은 진입 문턱 |
| 가격위치 | -1.5% ~ +2.5% | 넓은 진입 범위 |
| 기울기배수 | 1.0x | 느슨한 기울기 조건 |

**예상 성과**

| 항목 | 예상값 |
|------|-------|
| 승률 | 55~65% |
| 평균 수익 | +2~3% |
| 거래 빈도 | 월 5~10회 |
| 손익비 | 1.2:1 |

**특징**
- 거래 기회 많음
- 대형주 특유의 안정성 활용
- 슬리피지 적음 (유동성 풍부)

---

**적용 대상**
- 시총 1~10조 대형주
- 안정적인 중형주
- 대부분의 일반 종목

**파라미터 설정**

| 항목 | 값 | 설명 |
|------|-----|------|
| 최소상승일 | 3봉 | 적정한 확인 기간 |
| 변동성 상한 | 18% | 표준 변동성 |
| 손절률 | -3.5% | 표준 손절폭 |
| 익절률 | +5.5% | 표준 익절폭 |
| 점수 기준 | 7점 | 표준 문턱 |
| 가격위치 | -1% ~ +2% | 표준 범위 |
| 기울기배수 | 1.2x | 표준 기울기 조건 |

**예상 성과**

| 항목 | 예상값 |
|------|-------|
| 승률 | 60~70% |
| 평균 수익 | +2~4% |
| 거래 빈도 | 월 3~5회 |
| 손익비 | 1.5:1 |

**특징**
- 균형 잡힌 성과
- 대부분의 종목에 적용 가능
- 승률과 거래 빈도의 균형

---

### 4.3 고변동주 모드 (MODE 2)

**적용 대상**
- 테마주, 급등주
- 코스닥 소형주
- 변동성이 매우 큰 종목

**파라미터 설정**

| 항목 | 값 | 설명 |
|------|-----|------|
| 최소상승일 | 4봉 | 충분한 확인 |
| 변동성 상한 | 15% | 과도한 변동성 차단 |
| 손절률 | -3.0% | 빠른 손절 |
| 익절률 | +4.5% | 빠른 익절 |
| 점수 기준 | 8점 | 높은 진입 문턱 |
| 가격위치 | -0.5% ~ +1.5% | 좁은 진입 범위 |
| 기울기배수 | 1.5x | 엄격한 기울기 조건 |

**예상 성과**

| 항목 | 예상값 |
|------|-------|
| 승률 | 70~80% |
| 평균 수익 | +1.5~3% |
| 거래 빈도 | 월 1~3회 |
| 손익비 | 1.3:1 |

**특징**
- 높은 승률
- 거래 기회 적음
- 확실한 신호에서만 진입

## 5. 매수 조건 상세

매수 신호가 발생하려면 **다음 조건을 모두 충족**해야 합니다:

### 5.1 필수 조건 체크리스트

```
□ 1단계: 국면 확인
   → 반드시 1단계(급등세)이어야 함
   → 2~6단계에서는 매수 신호 없음

□ 2단계: 변동성 필터
   → (최근고점 - 최근저점) / 회귀선 < 모드별 상한
   → 과도한 변동성 구간 제외

□ 3단계: 기울기 확인
   → 기울기 > 강약기준 × 모드별배수
   → 충분한 상승 기울기 필요

□ 4단계: 추세 점수 확인
   → 추세점수 ≥ 2점
   → 연속상승일 ≥ 모드별 최소상승일

□ 5단계: 가격 위치 확인
   → 모드별 하한 ≤ (현재가-회귀선)/회귀선 ≤ 모드별 상한
   → 회귀선에서 너무 멀지 않은 위치

□ 6단계: 고저점 확인
   → 고점점수 ≥ 0 AND 저점점수 ≥ 0
   → 고점/저점 동시 하락 패턴 회피

□ 7단계: 점수 확인
   → 총점 ≥ 3 또는 상승점수 ≥ 모드별 기준
   → 충분한 상승 에너지 확인
```

### 5.2 진입 타이밍

```
최적의 진입 시점:
1. 6단계(박스하단)에서 에너지 축적
2. 2단계(상승세)로 전환
3. 1단계(급등세) 진입 + 점수 3+ 확인
4. 다음 봉 시가에 매수 진입

피해야 할 진입:
- 1단계가 이미 오래 지속된 경우
- 변동성이 급격히 확대된 직후
- 점수가 하락 추세인 경우
```

## 6. 매도 조건 상세

### 6.1 즉시 청산 조건

| 조건 | 설명 |
|------|------|
| 손절 도달 | 수익률 ≤ 모드별 손절률 |
| 익절 도달 | 수익률 ≥ 모드별 익절률 |

### 6.2 최소 보유일(2일) 이후 관리

| 조건 | 행동 | 우선순위 |
|------|------|----------|
| 국면 3(급락) 또는 4(하락) 전환 | 즉시 청산 | 1순위 |
| 총점 -2점 이하 | 추세 붕괴 판단, 청산 | 2순위 |
| 수익 0~1% + 점수 2점 이하 | 본전 청산 | 3순위 |
| 손실 -1.5% 이하 + 점수 2점 이하 | 약추세 손절 | 4순위 |

### 6.3 청산 시나리오별 대응

**시나리오 A: 정상 익절**
```
상황: 수익 +5.5% 도달 (일반주모드)
행동: 
- 전량 익절 또는
- 50% 익절 후 나머지는 트레일링
```

**시나리오 B: 손절**
```
상황: 손실 -3.5% 도달 (일반주모드)
행동: 즉시 전량 청산, 예외 없음
```

**시나리오 C: 국면 전환**
```
상황: 1단계 → 3단계(급락세) 전환
행동: 수익/손실 무관하게 즉시 청산
```

**시나리오 D: 본전 청산**
```
상황: 보유 3일차, 수익 +0.5%, 점수 2점
행동: 힘 빠짐 판단, 빠른 청산
```

## 7. 실전 매매 가이드

### 7.1 매수 진입 플로우

```
Step 1: 모드 선택
├── 삼성전자, 지수 → 대형주 모드
├── 일반 종목 → 일반주 모드
└── 테마주, 소형주 → 고변동주 모드

Step 2: 국면 모니터링
├── 6단계(박스하단) 관찰 → 에너지 축적 중
├── 2단계(상승세) 진입 → 주목
└── 1단계(급등세) 진입 → 매수 준비!

Step 3: 조건 확인
├── 점수 3점 이상? → Yes
├── 변동성 상한 이내? → Yes  
├── 연속상승일 충족? → Yes
└── 모든 조건 충족 → 매수 진입!

Step 4: 포지션 관리
├── 1차 매수: 50% 진입
├── 추가 상승 확인 → 2차 매수 30%
└── 신고가 돌파 → 3차 매수 20%
```

### 7.2 차트 읽기

**차트 좌측 상단 표시 해석**
```
[일반주] 급등세 7점
   ↑        ↑     ↑
  모드    국면   총점
```

**색상 해석**
- 빨강 배경 = 매수 적합 구간
- 주황 배경 = 보유 유지 구간
- 파랑/회색 배경 = 청산 또는 관망

### 7.3 분할 매매 전략

| 단계 | 조건 | 비중 | 누적 |
|------|------|------|------|
| 1차 | 1단계 진입 + 점수 3+ | 40% | 40% |
| 2차 | 점수 5+ 유지 | 30% | 70% |
| 3차 | 점수 7+ 또는 신고가 | 30% | 100% |

| 실수 | 문제점 | 해결책 |
|------|--------|--------|
| 1단계 아닌데 매수 | 승률 급락 | 국면 1 필수 확인 |
| 손절 미이행 | 큰 손실 | 기계적 손절 실행 |
| 모드 미스매치 | 잦은 신호/신호 없음 | 종목에 맞는 모드 선택 |
| 과거 성과 맹신 | 실전 괴리 | 충분한 테스트 후 적용 |

### 10.2 고급 팁

1. **점수 추세 관찰**: 절대 점수보다 점수의 방향성이 중요
2. **국면 체류 시간**: 1단계가 오래 지속되면 과열 주의
3. **복수 종목 운용**: 신호 분산으로 리스크 관리
4. **시간대 고려**: 장 초반/마감 직전 신호는 신뢰도 낮음

### 10.3 다른 신호와 조합

| 조합 | 시너지 효과 |
|------|------------|
| 골든돔 + 배틀존 | 두 신호 일치 시 고신뢰 |
| 골든돔 + 바로미터 | 게이지 방향 확인으로 필터링 |
| 골든돔 + 거래량 | 거래량 증가 동반 시 신뢰 상승 |
